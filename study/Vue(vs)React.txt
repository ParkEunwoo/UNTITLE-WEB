<Vue.js vs React.js>
1.공통점
   1. Virtual DOM으로 빠른 렌더링
   2. 경량 라이브러리
   3. Reactive Component
   4. Server Side Rendering
  
1.Vue의 장점
   1. Template 과 Render 함수을 모두 사용할 수 있는 옵션  (버전 2.x부터 render 함수도 지원함)
	1) html 파일에 마크업을 작성하는게 기본 설정
	2) {{ }} 형태로 데이터 바인딩
	3) 표준 html 태그 이외에 Directive 라는 특별 html atttritube를 이용해 template 기능 확장 가능
   2. 간편한 Syntax 와 프로젝트 설정
   3. 빠른 렌더링과 더 작은 용량

2.React의 장점
   1. JSX 를 사용하여 자바스크립트에서 DOM을 생성
   2. 큰 규모의 앱에 적합, 테스팅에 수월
	1) 리액트의 렌더링 시스템이 더 정밀한 구성이 가능
	2) 테스팅 도구와도 결합가능 -> 테스트하기 수월 & 유지보수가 더 쉬움
   3. 웹 앱과 네이티브 앱 개발에 모두 사용 가능
   4. 더 큰 개발자 생태계에서 오는 많은 레퍼런스와 도구들
